Frequently Asked Questions

Everything you need to know

Do I need any coding experience?

Not at all! Our program is specifically designed for people with zero coding experience. We teach you to build software using AI tools, which means you don't need to memorize syntax or learn traditional programming. If you can write clear instructions, you can learn to code with AI.

How much time do I need to commit?

The daily cohort runs 2 hours per day for 10 days (6-8 PM EST). The weekly cohort runs 2 hours on Saturdays for 10 weeks. We recommend an additional 1-2 hours of practice daily to maximize your learning.

However, practice is optional – we have plenty of time in class for you to learn deeply. The in-class time is designed to be comprehensive and hands-on.

Are there any additional costs beyond the tuition?

Yes, you'll need subscriptions to AI coding tools we use in the course:

  • Cursor IDE: $20/month (required)
  • Claude Pro (Anthropic): $20/month (recommended)
  • ChatGPT Plus or GitHub Copilot: $20/month (optional, but helpful)

Estimated minimum additional cost: $40-60/month during the course. Most students find these tools invaluable and continue using them after graduation.

What if I can't attend a session?

All sessions are recorded and available within 24 hours. You'll have lifetime access to all recordings, materials, and the community. Many students watch recordings even if they attended live.

What tools and technologies will I learn?

You'll learn to use cutting-edge AI coding assistants (Cursor, Claude, GPT-4), build with React and Node.js, work with databases like PostgreSQL, deploy to cloud platforms, and integrate AI APIs. We focus on the tools actual companies use.

How is this different from other bootcamps?

Traditional bootcamps spend 3-6 months teaching syntax memorization and debugging. We teach you to direct AI to write code for you, the same way senior engineers work today. This means you build production-quality applications in days, not months, with instructors from MIT, Stanford, Google, and Apple - some of whom have worked at Google, Apple, Meta, and other top companies using AI coding tools in real world applications.
